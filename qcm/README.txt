================================================================
        INDEX — MINI QUIZZES (ANSIBLE COURSE)
================================================================

This folder lists the per-module mini-quizzes shipped with the
Ansible course. Each quiz also exists as Slidev slides in the main
deck under pages/ and related imports.

================================================================

Files in this directory:

  module-04-inventaires.txt      - Inventories & hosts
  module-05-playbooks.txt       - Playbooks
  module-06-modules.txt         - Core modules
  module-07-variables.txt       - Variables
  module-08-templates.txt       - Jinja2 templates
  module-09-handlers.txt        - Handlers
  module-10-roles.txt           - Roles
  module-11-collections.txt     - Collections
  module-12-vault.txt           - Ansible Vault

================================================================

What was added to the course:

  - Mini-quizzes (3 questions) at the end of several modules
    -> Quick knowledge check
    -> Available as slides + plain text for handouts

  - Short hands-on labs (about 5–15 minutes) per module
    -> Immediate practice of the ideas just taught

  - Full "Debugging & troubleshooting" section in the deck
    -> Verbosity (-v / -vv / -vvv), debug module
    -> --check, --start-at-task, --step, --limit
    -> Common failures and fixes
    -> Debugging checklist

================================================================

Approximate extra slide volume: ~90 slides (order of magnitude),
including mini-quizzes, mini-labs, debugging chapter, and
transitions — exact count depends on Slidev merges.

================================================================

How to use the TXT quizzes:

1. Handouts
   - Share the TXT for the module you are teaching
   - Works offline; plain UTF-8 text

2. Quick evaluation
   - 3 questions ~= 2–3 minutes per module
   - Answer key included in each file for self-check

3. Revision
   - Learners can re-read between sessions
   - Structured question / answer layout
