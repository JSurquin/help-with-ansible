# PREREQUISITES AND ENVIRONMENT 📋

### Recommended knowledge

- **Linux/Unix**: Basic command-line skills
- **Development**: Basic concepts (any language)
- **Networking**: Basic understanding of network concepts
- **YAML**: Basic syntax (covered in depth during the training)

### Technical environment

- **Docker Desktop 2026** or Docker Engine on Linux
- **Ansible 2026** (guided installation during the training)
- **Terminal/Shell**: Bash, Zsh, or PowerShell
- **Editor**: VS Code recommended with Docker and Ansible extensions
